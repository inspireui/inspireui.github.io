Customization

-- Last update 15/01/2019 --

1. Change main color

Open Common/AppConfig.json and update these values, since v1.2.0 this could be updated via the UI of CeBuilder.

  "MainColor": "#d45547"

2. Change the app name, app icon and splash screent

Open app.json and replace these values:

"name": "CeNews",
....
"icon": "./assets/icons/app-icon.png",
"loading": {
    "icon": "./assets/icons/loading.png",
},
"splash": {
      "image": "./assets/icons/loading.png",
},

3. Switching the CeNews to BuzzApp template.

Open the App/Common folder:

  • Open the AppConfig_Buzzy.json and copy the content to AppConfig.json (this is used for main loading for the app)
  • Open the Config_Buzzy.js and copy the content to Config.js file.
